Random Number Generator - for the Eurojackpot lottery

About

One friday afternoon back in November, I heard about a jackpot of almost 100million euros in the Eurojackpot lottery. I was going through some vanilla JavaScript tutorials at that time and even though I am usually not a regular lottery player, I had the idea to combine the lottery event with my JavaScript learnings, by creating a random number generator. My biggest learning here was, using the JavaScript Set constructor to create sets of random numbers without any repeating numbers.
